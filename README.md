# PNU QuestMate V10 Semester 100 Rewards

## 핵심 변경점
- 스탬프 목표를 25개에서 **한 학기 100개**로 확대
- 하루 1개 제한은 유지
- 20/40/60/80/100개 단계형 리워드 체계 적용
- 80개 이상: 비교과 마일리지 신청 대상
- 100개: 학기 우수참여자 인증서 + 학기말 우수참여자 선발 후보
- 상위 참여자 보상은 장학금이라는 표현보다 활동지원금/학습지원금/우수참여자 장려금 형태로 제안

## 발표용 빠른 검증
실제 운영 기준은 학습형 3시간 체류입니다.
발표 중 짧게 확인할 때만 주소 뒤에 `?presenter=1`을 붙이면 10초 검증 모드가 활성화됩니다.
